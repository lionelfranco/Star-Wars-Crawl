An unexpected journey


Our story takes place in Belgium. A young man, Lionel, is about to start a coding formation at BeCode.

When he arrives, he automatically realise that friendship is about to be the main word. First day, the coach made his speech. Lionel also initialised his terminal so he can use it for work later.

Days are passing fast. Lionel learn a new thing everyday. For example, he now is able to code a page that show some sort of a Star-Wars generic (obviously)...

What will happen next ? Only Leny knows...
